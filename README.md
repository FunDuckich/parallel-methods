# Решения лабораторных работ
**Предмет:** Параллельная обработка и анализ данных

Этот гайд поможет тебе быстро скомпилировать и запустить программы из любой папки проекта. 

## 🛠 Что нужно для запуска
1. **Компилятор C++ (GCC/MinGW)** — должен быть установлен в системе.
2. **Настроенный Path** — команда `g++` должна работать в терминале.
3. **Библиотека OpenMP** — она встроена в компилятор, мы просто включаем её флагом.
4. **MPI** — для 6 лабы нужны команды `mpic++` и `mpiexec`.

## 📁 Структура проекта
Все задания разложены по папкам:
`номер_лабы / номер_задания`

Например:
* `1/1` — Лабораторная 1, задание 1
* `2/3` — Лабораторная 2, задание 3
* `3/7` — Лабораторная 3, задание 7

---

## 🚀 Как запустить любое задание

Для запуска используем терминал (PowerShell или CMD).

### Общий алгоритм:
1. Открой терминал в корневой папке проекта.
2. Перейди в папку нужного задания командой `cd`.
3. Скомпилируй код с флагом `-fopenmp`, а для 6 лабы через `mpic++`.
4. Запусти созданный `.exe` файл.

Если `mpiexec` не находится в текущем терминале, открой новый PowerShell или используй полный путь:
`& "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe"`.

### Примеры для твоих папок:

#### Лаба 1, Задание 1
```bash
g++ -fopenmp 1/1.cpp -o 1/1.exe
.\1\1.exe
```

#### Лаба 2, Задание 3
```bash
g++ -fopenmp 2/3.cpp -o 2/3.exe
.\2\3.exe
```

#### Лаба 3, Задание 7
```bash
g++ -fopenmp 3/7.cpp -o 3/7.exe
.\3\7.exe
```

#### Лаба 6, Задание 1
```bash
mpic++ 6/1.cpp -o 6/1.exe
& "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe" -n 4 .\6\1.exe
```

#### Лаба 6, Задание 3
```bash
mpic++ 6/3.cpp -o 6/3.exe
& "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe" -n 2 .\6\3.exe
```

#### Лаба 6, Задание 6
```bash
mpic++ 6/6.cpp -o 6/6.exe
& "C:\Program Files\Microsoft MPI\Bin\mpiexec.exe" -n 2 .\6\6.exe
```
